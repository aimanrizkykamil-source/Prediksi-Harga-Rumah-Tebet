# Prediksi-Harga-Rumah-Tebet
Aplikasi prediksi harga rumah di wilayah Tebet menggunakan algoritma Random Forest Regressor. Aplikasi ini menerima input luas bangunan, luas tanah, jumlah kamar tidur, kamar mandi, dan garasi, lalu menampilkan estimasi harga rumah dalam Rupiah melalui antarmuka Gradio.
